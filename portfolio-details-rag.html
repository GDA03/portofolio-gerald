PDF-based Chatbot with Local Retrieval-Augmented Generation (RAG)

Project Overview – NLP, Vector Database, and LLM Integration

Berhubung Capstone Project di Bangkit Academy saya adalah mengenai Natural Language Processing (NLP) pada machine learning, maka saya mengembangkan sebuah chatbot berbasis dokumen PDF dengan pendekatan Local Retrieval-Augmented Generation (RAG) yang di publikasikan pada Open Journal System dari Universitas Lampung. Proyek ini dirancang untuk meningkatkan efisiensi pencarian informasi dalam dokumen secara lokal—tanpa ketergantungan pada layanan cloud—sehingga lebih aman, hemat biaya, dan menjaga privasi data pengguna.

Tujuan dan Pendekatan

Tujuan utama proyek ini adalah membangun sistem chatbot yang mampu:

  • Memahami isi dokumen PDF secara mendalam.
  • Menjawab pertanyaan pengguna dengan relevansi tinggi.
  • Beroperasi secara lokal untuk mendukung lingkungan terbatas koneksi atau kebutuhan privasi tinggi.

Untuk mendukung sistem ini, digunakan beberapa komponen teknis utama:

  • ChromaDB sebagai vector database untuk menyimpan representasi embedding teks
  • nomic_embed_text sebagai model embedding untuk menghasilkan representasi vektor yang akurat
  • Ollama sebagai Local LLM Manager yang mengatur proses inferensi jawaban

Metodologi Pengembangan

Metode yang digunakan adalah Rapid Application Development (RAD), dengan tahapan:

  • Planning – Identifikasi kebutuhan pengguna dan ruang lingkup fungsionalitas chatbot
  • Design – Perancangan arsitektur sistem dan antarmuka pengguna
  • Implementation – Integrasi vector store, model embedding, dan LLM
  • Testing – Evaluasi performa dan kualitas jawaban menggunakan metrik ROUGE-L

Hasil dan Evaluasi

  • Skor ROUGE-L sebesar 0.85, menunjukkan relevansi yang tinggi antara jawaban chatbot dan konteks dari dokumen.
  • Sistem terbukti efisien dalam pencarian informasi, serta mampu menjaga keamanan data karena seluruh proses berlangsung secara lokal.
  • Beberapa keterbatasan masih ditemukan, seperti dukungan bahasa yang terbatas dan variasi waktu respon tergantung spesifikasi perangkat pengguna.

Kesimpulan

Proyek chatbot berbasis Local RAG ini menunjukkan potensi sebagai solusi efisien, privat, dan scalable untuk kebutuhan pencarian informasi dalam dokumen. Ke depan, pengembangan diarahkan pada peningkatan dukungan multi-bahasa, optimasi kecepatan inferensi, serta peningkatan pengalaman pengguna agar lebih adaptif terhadap berbagai skenario industri maupun kebutuhan riset.
